This is a relative piano: "Biano". The name comes from "Binary" and "Piano".
Biano works by pressing a keyboard key or by touching a button.
Initially, the note is "C4". If you want to play "D4", you press the "W" key. If you want to play "G4", you press the "Q", "W", "E" keys. To repeat the same note, press the "Q", "A" keys.
The keyboard push-threshold slider adjusts the delay of the piano, while the number-of-keyboard slider adjusts the number of keys.
There is also a context menu that appears when you right-click on a key, which allows you to change the keyboard layout.
Enjoy!
